External documentation

General C++

  • The C++ Standard : A web version generated for the LaTeX draft sources. Not an ISO publication.
  • CPlusPlus.com : A great ressource for C++/C++11. It has tutorials and documentation on the standard library. The documentation is a bit outdated though: C++14/17 are not yet taken into account.
  • Compiler explorer : Lets you try out various compilers and study the assembly code related to your C++ snippet.

Floating point arithmetic

  • Comparing Floating Point Numbers, 2012 Edition : A very nice and to the point introduction to the subject of floating point comparison by Bruce Dawson. Also includes a lot of links to earlier articles by the same author.
  • What Every Computer Scientist Should Know About Floating-Point Arithmetic : A very in-depth explanation of floating point arithmetic. The article is long and more academic but has a lot of interesting information. The author is David Goldberg.

External libraries

  • Programming with Gtkmm 3 : The reference manual, by Gnome, for the stable release of the Gtkmm library. It has explanations and examples, but is incomplete.
  • Gtkmm developper documentation : API documentation. Essentially a Doxygen output of the project.
  • Gtk+ Reference Manual (in C) : Sometimes Gtkmm's documentation is too incomplete to work with and diving into a good Gtk+ manual is the only option. This is the official manual.

Make

  • GNU make official documentation : Official GNU documenration for GNU make. Very complete but sometimes hard to follow.
  • Swarthmore College make examples : A collection of basic examples. Very understandable.
