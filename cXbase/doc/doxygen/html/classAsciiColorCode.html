Detailed Description

An ASCII code representing a color.

An ASCII color code is an ASCII character that is used to represent a color (for example in a terminal, or a text file). Not all ASCII characters can be used to represent a color. In fact, only letters (lower or upper case) and the space character can be used.

For example, these are some of the ways the color Black can be represented:

  • the character 'B';
  • the character 'b';
  • the character 'x';
  • the character ' '.

However, the last two examples are not recommended for readibility purposes.

Invariant
An ASCII color code is always a letter (lower or upper case) or a space character.
