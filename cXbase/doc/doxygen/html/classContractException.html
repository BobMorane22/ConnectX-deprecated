Detailed Description

Basis for design-by-contract type exceptions.

Defines a common anatomy for design-by-contract type exceptions. Each of these exceptions must document four key components: the file and line where the exception is thrown, the expression throwing the exception and the message carried by the exception. All design-by-contract exceptions must at least inherit from this class.
