Detailed Description

class GameBoard

Connect X GameBoard.

A Connect X GameBoard consists mainly of a Grid, or a 2D Disc array and dimensionnal information about the grid. In this document, grid and GameBoard will be used interchangeably. The GameBoard also support features to check itself for a winner if additionnal information is supplied (like the in a row value for the game).

Invariant
The number of rows exceeds the minimal number of rows: 6
The number of rows is less than the maximal number of rows: 64
The number of columns exceeds the minimal number of columns: 7
The number of columns is less than the maximal number of columns: 64
Note
If you want dimensions smaller than 6 by 7 or larger than 64 by 64, you must recompile the cXbase library after having modified the values for private data members NB_ROWS_MIN, NB_COLUMNS_MIN, NB_ROWS_MAX and NB_COLUMNS_MAX to your needs. Note that the library has only been officially tested for default values.
3D GameBoards are not supported.
Examples:
Classic Connect 4 minimal implementation.

bool GameBoard::isColumnFull ( const Column &  p_column) const

Checks if a specific Column is full.

If Discs (other than the NO_DISC specifier) are stacked all the way up to the last Column's position, the Column is considered full. Concretely:

*
*   Example:
*
*      |   | <- Not full, no Disc here...
*      |   |
*      |   |
*      |   |
*      |   |
*      | R |
*
*
*      |   | <- Not full yet, still no Disc here...
*      | R |
*      | N |
*      | R |
*      | B |
*      | R |
*
*
*      | N | <- Now you got it! The Column is full.
*      | R |
*      | N |
*      | R |
*      | B |
*      | R |
*
*
Parameters
[in]p_columnThe Column where to insert the Disc.
Precondition
The Column number is between 0 and the maximum Column number for the GameBoard.
Returns
true if the Column passed as an argument is full, false otherwise.

Checks for a win at a specified Position.

Checks from the specified Position if a combination of p_inARow consecutive Discs matching the Disc at p_positionLastDiscPlaced ispresent either horizontally, vertically or diagonally. Be aware that it is NOT the responsablity of this method to know who won. In other words, it only checks if someone won.

Position GameBoard::placeDisc ( const Column &  p_column,
const Disc &  p_disc 
)

Places a Disc in a specific Column.

Just like in the classic Connect 4 game, the Disc is inserted in a specific Column and sets to the bottommost available Position in the Column.

*
*   Example:
*
*      | N | <- Last placed disc (unless already full, see below)
*      | R |
*      | N |
*      | R |
*      | N |
*      | R | <- First placed Disc.
*
*

If the Column is already full, the action is ignored (the disc is not placed!) ad the top most position is returned. It is a good idea to use the isColumnFull() method before trying to place a Disc.

Parameters
[in]p_columnThe Column where to insert the Disc.
[in]p_discThe Disc to insert.
Returns
The Position where the Disc has been placed, if the Column specified as an argument is not full, or the Column's top most Position otherwise.
Precondition
The Column number is between 0 and the maximum Column number for the GameBoard.
The Disc that is placed has a ColorAsciiCode other than the space character (' ').

Stream insertion operator.

A GameBoard can be printed to text-mode using this operator. Only the grid and its underlying Discs are printed with Row and Column numbers. The character "|" is used to separate different columns.

For example:

// A disc, red in this case:
Disc RED_DISC{Color{Name{"Red"}, AsciiColorCode{'R'}}};
// A classic Connect 4 board:
GameBoard aBoard;
// Fill first Column:
aBoard.placeDisc(Column{0}, RED_DISC);
aBoard.placeDisc(Column{0}, RED_DISC);
aBoard.placeDisc(Column{0}, RED_DISC);
aBoard.placeDisc(Column{0}, RED_DISC);
aBoard.placeDisc(Column{0}, RED_DISC);
aBoard.placeDisc(Column{0}, RED_DISC);
aBoard.placeDisc(Column{0}, RED_DISC); // Oops! No effect: already full!
// Put three Discs in second:
aBoard.placeDisc(Column{1}, RED_DISC);
aBoard.placeDisc(Column{1}, RED_DISC);
aBoard.placeDisc(Column{1}, RED_DISC);
// Print it to console:
std::cout << aBoard;

will result in the following text string inserted:

*
*       5 | R |   |   |   |   |   |   |
*       4 | R |   |   |   |   |   |   |
*       3 | R |   |   |   |   |   |   |
*       2 | R | R |   |   |   |   |   |
*       1 | R | R |   |   |   |   |   |
*       0 | R | R |   |   |   |   |   |
*           0   1   2   3   4   5   6
*
*
Parameters
[in]p_fluxThe stream in which to insert.
[in]p_gameBoardThe GameBoard to insert.
