Detailed Description

A colored Disc object.

A Disc can be seen as a kind of colored token. It's only data member is a Color.

See also
Color

std::ostream& operator<< ( std::ostream &  p_flux,
const Disc &  p_disc 
)
friend

Stream-insertion operator.

Inserts the Disc formatted in text mode into the stream. To do so, it uses the Disc Color's AsciiColorCode. It adds one space character to the left and one space character to the right of the AsciiColorCode char value. For example, if you define a Disc as such:

*    Disc aDisc{Color{Name{"Red"}, AsciiColorCode{'R'}}};
*

You will get the following string inserted into the stream:

*    " R "       // Without the quotes
*

So if you write this:

*    std::cout << "|" << aDisc << "|";
*

You will see this to the console:

*    "| R |"    // Without the quotes
*
Parameters
[in]p_fluxThe stream in which to insert.
[in]p_discThe Disc to output to the stream.
