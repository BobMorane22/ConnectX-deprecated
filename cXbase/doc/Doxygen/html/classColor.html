Detailed Description

A Color refers to a color Name paired with an ASCII char AsciiColorCode to be displayed in text mode. When a Color object representing the absence of color is needed, two values are made available to the developper: NO_COLOR (Name) and NO_COLOR_ASCII_CODE (AsciiColorCode).
