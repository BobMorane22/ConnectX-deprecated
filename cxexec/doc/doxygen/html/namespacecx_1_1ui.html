std::string cx::ui::buildGdkColorString ( const cxutil::Color &  p_localColor)

Builds a string of the form rgba(r, g, b, a) where r , g b and a represent the four RGBA components for the color. This function is useful to build Gdk::RGBA or Gdk::Color (deprecated) objects from a cxutil::Color type, which does not deal with strings.

Parameters
p_localColorThe cxutil::Color to be converted to a Gdk compatible string.
Returns
The Gdk compatible string representing the color passed as an argument.
